# Ejercicio 5: Suma de Números Enteros
## Descripción
Este programa funciona como una calculadora básica de una sola operación. Solicita al usuario dos valores numéricos, realiza la conversión de tipo de dato (de cadena a entero) y ejecuta una operación aritmética de suma, presentando el resultado final de manera clara.

## Conceptos Aplicados
* **Casting de Datos (`int`)**: Transformación de la entrada estándar (`string`) a un tipo de dato numérico entero para permitir operaciones matemáticas.
* **Operadores Aritméticos**: Uso del operador `+` para la adición de valores.
* **Almacenamiento en Variables**: Uso de variables (`x`, `y`, `z`) para organizar el flujo de entrada, proceso y salida de datos.

## Ejemplo de Ejecución
* **Entrada 1**: 10
* **Entrada 2**: 15
* **Operación**: 10 + 15
* **Resultado**: 25
